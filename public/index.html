Installe l'app ntfy (Android/iOS), abonne-toi à un sujet au nom unique, puis indique-le ici. Le serveur t'enverra un récap des relances. Astuce : utilise un nom long et difficile à deviner.

Sujet à ajouter dans l'app ntfy : —

Affiche une notification système quand l'app est ouverte et que des relances sont en attente.

Étiquettes

Ta liste d'étiquettes, assignables à chaque candidature et utilisables comme filtre.

Choisis une couleur dans la palette avant d'ajouter. Pour changer la couleur d'une étiquette existante, clique sur sa pastille de couleur.

Remplissage auto depuis l'offre

—

Dans une candidature, colle le lien d'une offre puis clique sur « ✨ Remplir » : l'app lit les données structurées de la page (gratuit). Pour les sites qui n'en fournissent pas, tu peux activer l'extraction par IA (Claude) en définissant une clé API.

Pour activer l'IA : crée une clé sur console.anthropic.com, puis lance l'app avec la variable d'environnement ANTHROPIC_API_KEY : 

# PowerShell (Windows)
$env:ANTHROPIC_API_KEY="sk-ant-..."
npm start

Le « coller le texte de l'offre » fonctionne aussi sans clé dès que l'IA est activée, et reste le meilleur recours pour LinkedIn/Indeed.

À propos

Application 100% locale. Tes données sont stockées sur ton ordinateur (base data/gestion.db et fichiers dans uploads/). Seule la fonction « Remplir depuis l'offre » contacte Internet (la page de l'offre, et l'API Claude si tu l'actives).
